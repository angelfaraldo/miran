miran
=====

mir and music analysis tools in python. Mostly tonality related stuff for EDM.

Documentation, including installation and usage information: http://angelfaraldo.github.io/miran/

Dependencies:

* numpy
* scipy
* future
* pandas
* music21
* madmom
* librosa
